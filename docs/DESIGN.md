## Visual Theme
A quietly-confident developer-tools aesthetic built on a warm-cream editorial canvas (`#f7f7f4`) rather than the typical dark IDE atmosphere. Near-black warm ink (`#26251e`) carries both body and display text; display type sits at weight 400 with negative letter-spacing for a magazine feel instead of a loud tech voice. A single brand accent — **Cursor Orange (`#f54e00`)** — is reserved for primary CTAs and the wordmark. Depth is **hairline-only**: no drop shadows or elevation tiers — cards float above the canvas via 1px hairline borders and subtle white-on-cream contrast. Generous 80px section rhythm. **JetBrains Mono** is used on every data/code surface (IP addresses, device labels, lengths, DORI specs) — a natural fit for a technical ICT planning tool. A pastel status palette (peach/mint/blue/lavender/gold) marks states without heavy color.
